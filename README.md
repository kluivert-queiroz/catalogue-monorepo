# Catalogue
How to run
```shell
git clone https://github.com/kluivert-queiroz/catalogue-monorepo.git

cd catalogue-monorepo

docker-compose up
```

You'll have the API at `localhost:3000` and the UI at `localhost:3001`

Technologies used:
- NodeJS
- React
- GraphQL
- Subscriptions w/ GQL
- MongoDB
